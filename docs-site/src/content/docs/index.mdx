## Key Capabilities

<CardGrid>
  <Card title="Live Transcription" icon="open-book">
    Real-time speech-to-text with speaker attribution, multi-language support,
    live translation, PII redaction, and custom vocabulary.
  </Card>
  <Card title="Meeting Assistant" icon="rocket">
    AI-powered Strands agent chat interface with built-in tools, Knowledge Base
    integration, Guardrails, model selection, and custom prompts.
  </Card>
  <Card title="Transcript Summarization" icon="document">
    Automatic meeting summaries, action items, and key decisions generated by
    Amazon Bedrock during and after meetings.
  </Card>
  <Card title="Voice Assistant" icon="microphone">
    Natural voice interaction with Nova Sonic, ElevenLabs TTS, and optional
    Simli avatar for face-to-face AI assistance.
  </Card>
</CardGrid>
